# DatingApp

This repo contains a starter **dating app** web project.

## Where the code lives

- **Web app**: `dating-app/` (Next.js + TypeScript + Tailwind)

## Run it locally

```bash
cd dating-app
npm run dev
```

Then open `http://localhost:3000`.

## Next coding steps (recommended order)

1. **Define the MVP**
   - Onboarding: create/edit profile (name, age, bio, photos)
   - Discover: swipe cards (like / pass)
   - Matches: list mutual likes
   - Chat: basic 1:1 messaging for matches

2. **Add a real backend**
   - Database (Postgres recommended)
   - Auth (email/password or OAuth)
   - APIs for profiles, swipes, matches, messages

3. **Ship safely**
   - Input validation, rate limits, moderation/reporting
   - Image upload + processing
   - Privacy controls + blocking
